Нужно протестировать, что распределение показов по баннерам работает правильно в соответствии со следующим соотношением:

Баннер B1 - 10%  
Баннер B2 - 40%  
Баннер B3 - 25%  
Баннер B4 - 25%


Заполнить в docker-compose.yml переменные окружения
`
API_KEY
PUBLISHER_ID
ZONE_ID
`

Сборка образа  
`docker-compose up -d --build`  

Запуск тестов  
`./run_tests.sh`
